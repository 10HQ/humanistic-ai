# Contributing to Humanistic AI

> *"不是因为它能赢，而是因为没有它会更糟。"*

---

## How to Contribute

### 1. Discussion
- Open a [Discussion](https://github.com/10HQ/humanistic-ai/discussions) to share ideas, critiques, or resources
- We welcome **constructive criticism** — the manifesto has survived 4 rounds of it

### 2. Issues
- Report bugs, logical errors, or suggest improvements
- Use the provided templates

### 3. Pull Requests
- Fork the repo
- Create a feature branch
- Submit a PR with clear description
- All contributions are under CC BY-SA 4.0

## Values

- **Critical thinking**: We welcome criticism, not flattery
- **Process justice**: Disputes resolved through discussion
- **Substantive red lines**: Non-negotiable ethical baselines
- **Uncertainty**: We acknowledge we might be wrong
- **Praxis**: Theory must become code

---

## Code of Practice

1. **Be precise**: Vague claims need evidence
2. **Be honest**: Don't overstate capabilities
3. **Be constructive**: Critique with alternatives
4. **Be open**: Share your reasoning
5. **Be humble**: You might be wrong
