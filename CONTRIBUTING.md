# 贡献指南

感谢你对人文AI项目的关注。

## 如何参与

### 讨论
- 在 [Discussions](https://github.com/10HQ/humanistic-ai/discussions) 中分享你对人文AI的思考
- 对宣言提出批评和建议
- 分享相关的研究、案例、资源

### 提Issue
- 报告宣言中的逻辑漏洞或事实错误
- 提出新的理论框架或技术实现方向
- 分享人文AI在实际场景中的应用案例

### 提PR
- 修复文档错误
- 补充理论框架
- 改进技术实现
- 翻译为其他语言

## 项目结构

```
manifesto.md           ← 宣言（核心文本）
theory-framework.md    ← 理论框架
technical-roadmap.md   ← 技术实现路径
cognitive_distortion_detector.py  ← CBT认知扭曲检测器
concept_clarifier.py   ← 概念澄清引擎
ethics_checker.py      ← 伦理检查层
humanistic_ai.py       ← 引擎主模块
```

## 价值观

- **批判性思维**：欢迎批评，不欢迎奉承
- **过程正义**：所有争议通过讨论解决
- **实质底线**：不得违反基本伦理底线
- **不确定性**：承认我们可能是错的
