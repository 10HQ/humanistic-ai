# 人文AI宣言

> *"技术决定我们能做什么，人文决定我们应该做什么。"*

---

## 什么是人文AI？

人文AI不是给AI加上一层人文的皮。它是一种根本不同的AI范式。

**人文AI = 知识深度 × 洞察力 × 实践智慧**

## 三大支柱

1. **哲学素养**：追问本质、澄清概念、逻辑严谨、价值反思
2. **心理学素养**：共情理解、行为洞察、识别痛苦、促进成长
3. **实践智慧**：情境判断、伦理敏感、行动导向、审慎谦逊

## 为什么需要人文AI？

- **技术的傲慢**：算法撕裂共识、制造焦虑、瓦解真实
- **人文的缺席**：没有人在思考AI应该把人类带向何方
- **智慧的饥荒**：信息爆炸但智慧稀缺
